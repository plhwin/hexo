## 概述
jinfo可以用来查看正在运行的java运用程序的扩展参数，甚至支持在运行时动态地更改部分参数，他的基本使用语法如下：
>jinfo -< option > < pid >
<!--more-->
其中option可以为以下信息：
* -flag< name >: 打印指定java虚拟机的参数值。
* -flag [+|-]< name >：设置或取消指定java虚拟机参数的布尔值。
* -flag < name >=< value >：设置指定java虚拟机的参数的值。
